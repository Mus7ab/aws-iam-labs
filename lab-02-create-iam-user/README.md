# Lab 02 – Create IAM User

**Status:** 🔄 In Progress

## Objective

Create an IAM user with AWS Management Console access and understand how IAM users differ from the Root User.

## Tasks Planned

* Create an IAM user
* Enable AWS Management Console access
* Assign permissions
* Test login using the IAM user
* Enable MFA on the IAM user

## Evidence

Screenshots will be stored in the `screenshots/` folder.

## Key Learning

IAM users provide individual identities and should be used instead of the Root User for daily AWS activities.
