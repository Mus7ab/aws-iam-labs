# Lab 01 – Root Account Security & MFA

**Status:** ✅ Complete

## Objective

Secure the AWS root account using MFA and review account security best practices.

## Tasks Completed

* Enabled MFA on the AWS root account
* Verified root email and phone number
* Reviewed account security settings
* Created an IAM Administrator user
* Enabled MFA on the IAM Administrator user
* Stopped using the root account for daily administration

## Evidence

Screenshots are available in the screenshots folder.

## Key Learning

The root account has unrestricted access to all AWS services and resources. It should only be used for account-level tasks and must always be protected with MFA.
