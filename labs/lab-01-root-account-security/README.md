# Lab 01 – Root Account Security & MFA

## Objective

Secure the AWS root account using MFA and review account security best practices.

## Tasks Completed

- Enabled MFA on the AWS root account
- Verified root email and phone number
- Reviewed account security settings
- Avoided daily use of the root account

## Evidence

Add screenshots in the `screenshots` folder.

## Key Learning

The root account has unrestricted access to all AWS services and resources. It should only be used for a few account-level tasks and must always be protected with MFA.
