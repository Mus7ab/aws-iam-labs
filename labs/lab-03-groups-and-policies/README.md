# Lab 03 – IAM Groups and Permission Management

**Status:** 🔄 In Progress

## Objective

Learn how to manage permissions using IAM Groups and AWS Managed Policies.

## Tasks Planned

* Create IAM Groups
* Attach AWS Managed Policies
* Create test IAM Users
* Add users to groups
* Verify inherited permissions
* Understand permission management best practices

## Evidence

Screenshots will be stored in the `screenshots/` folder.

## Key Learning

IAM Groups simplify permission management by allowing permissions to be assigned to groups instead of individual users.
